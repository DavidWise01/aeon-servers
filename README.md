## Aeon Architecture (ABD Law Engine)

```
A = ANCHOR   — Containment. Finds definitional/boundary match in KB.
B = WITNESS  — Modulation. Finds contrasting voice in different KB file.
C = LAW      — Synthesis. A + B → coherent statement.
```

States: `A→B`, `B→C`, `C→A`, `A→C`, `C→B`, `B→A`, Home, Forward.  
FNV-1a witness hash. Auto-seed: copies `kb/` to `/data/kb` on first boot.
